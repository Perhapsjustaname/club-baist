# CLUB-BAIST Golf Course Management

Welcome to the CLUB-BAIST web application. This project is a specialized golf course booking system developed using Blazor and MS SQL Server. 


## Tech Stack

* **Frontend**: Blazor Web App (.NET 8/9)
* **Backend**: ASP.NET Core
* **Database**: Microsoft SQL Server
* **Authentication**: Custom Auth with BCrypt.Net-Next for password hashing
